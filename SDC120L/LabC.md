# Lab C — From Variables to Objects
## Objective
Introduce objects as containers for state.

## Activities
Students create a class called ```Item``` or ```User``` with:
- At least 3 fields
- A constructor
- A ```toString()``` method

Then:
- Create multiple objects
- Store them in an ```ArrayList```
- Print all objects

## Requirements
- No menus required
- Focus on object creation and storage

## Completion Check
- Objects persist across method calls
- Output shows multiple distinct objects

## Skills Practiced
✔ Class creation

✔ Object instantiation

✔ Encapsulation

✔  Method writing

✔  Collections
