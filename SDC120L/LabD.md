# Objective
Combine menus, logic, and objects into one program.

## Activities
Students extend a menu program to:
- Create objects from user input
- Store them in an ```ArrayList```
- List stored objects
- Prevent invalid actions

## Requirements
- Input validation required
- No crashes allowed
- Clean loop exit

## Completion Check
- User can add and list objects repeatedly
- Program remains stable under bad input
